Mac 安装 Apache Tomcat

2019-02-19 | Configuration

1.前往官网下载 Tomcat (可以选择zip或者tar.gz格式)

2.添加环境变量路径

1.将下载的压缩包解压到某个指定目录下
2.打开终端，输入命令 $ sudo pico .bash_profile
3.添加解压文件下 /bin 文件夹所在路径：（可以将文件夹拖入终端获取文件夹路径）

4.编辑完成后，保存并退出：Ctrl+X -> Y -> 回车
5.在终端中使用 $ cd 命令切换到 /bin 文件夹

3.启动Tomcat

1.输入命令 $ sudo ./startup.sh ，如出现下图所示则启动 Tomcat 成功

2.打开浏览器，在搜索框中输入网址 http://localhost:8080 ，如果能正常打开tomcat首页，说明 Tomcat 配置成功

4.关闭Tomcat命令是 $ sudo ./shutdown.sh
